/* 这是关于申请夏令营活动的说明*/


强调：！！！！！！！！一定一定不要怕麻烦！！！！！！！

报名夏令营是你选择学校（主动）
夏令营录取是学校选择你（被动）
你被录取了，决定去不去（主动）

可以多选择几所学校，由于民大是双非学校，很多好的学校会拒绝我们的学生。虽然看到那些夏令营通知的要求与自己不符，还是可以尝试投一
下（不要怕麻烦，万一人家看上你了呢，不要错过任何机会）。


每年的4月份开始，各高校的夏令营活动会陆陆续续下发通知，请大家留意。

提前选好自己心仪的研究生院校。

此处提醒：可以通过很多途径获得各高校夏令营活动消息：
（1）直接登录意向院校的相关学院的官网，时刻留意夏令营通知，夏令营通知出得早的有四月份就开始了，晚的有六月份出通知（少数学校7月份出夏令营通知）。
（2）下载考研帮（手机app），在app中日历菜单-》研讯-》置顶的那条信息就是夏令营通知。

备注：考研帮的信息仅供参考，要了解详情还需去相关院校的相关学院查看。
